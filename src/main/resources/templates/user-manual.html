1. Getting Started

Welcome to the Quiz Application! This interactive learning platform allows you to test your knowledge across various subjects with customizable quiz experiences.

Accessing the Application

  • Open your web browser and navigate to the application URL
  • The application is fully responsive and works on desktop, tablet, and mobile devices
  • Supports modern web browsers with JavaScript enabled

Language Support

The application supports multiple languages including English, Hungarian, and Vietnamese. You can change the language using the user menu in the top navigation.

2. Quiz Categories

The application organizes quizzes into different categories to help you find relevant content easily.

Category Features

  • Categories are displayed in an attractive grid layout with color-coded cards
  • Responsive design adapts to different screen sizes (1 column on mobile, 2 on tablet, 3 on desktop)
  • Interactive hover effects provide visual feedback
  • Click "Start Learning" to explore quizzes in that category

5. Quiz Results and Review

After completing a quiz, you'll receive comprehensive results and the opportunity to review your performance.

Score Summary

  • Large circular display showing your percentage score
  • Clear breakdown of correct vs. total questions
  • Gradient background for visual appeal

Wrong Answers Review

The application automatically compiles all questions you answered incorrectly for review:

  • Complete list of incorrect questions
  • Shows the correct answers
  • Displays your selected answers
  • Detailed explanations when available

Explanation Modal

For wrong answers with explanations, a detailed modal provides:

  • Detailed explanation of the correct answer
  • Continue button to proceed through review
  • Close option to exit explanation

üìö Learning Tip: Use the wrong answers review to identify knowledge gaps and focus your future study efforts.
